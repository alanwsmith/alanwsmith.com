```python
data = '''Alfa
Bravo
Charlie
Delta
Echo
Foxtrot'''

data_things = data.split("\n")

line = ", ".join(data_things)

print(line)
```

```bash
# Outputs
Alfa, Bravo, Charlie, Delta, Echo, Foxtrot
```
