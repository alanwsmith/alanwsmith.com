This is how I ignore the `node_modules` directory when running `tree`. 


```
tree -I node*
```

The `-I` says to ignore the specific pattern and the `node*` does the rest. Running `tree` without the ignore pattern on a basic nextjs install results in ~16K lines. Using the ignore pattern, you get this:


```
.
├── LICENSE
├── README.md
├── next.config.js
├── package-lock.json
├── package.json
├── pages
│   ├── _app.js
│   ├── api
│   └── index.js
├── public
└── styles
    ├── Home.module.css
    └── globals.css
```
