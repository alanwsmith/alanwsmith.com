title: Use A Specific AWS Credentials Profile With boto3.client
type: post
---

Use `setup_default_session` to use another profile with a `boto3.client()` call:


```python
import boto3

boto3.setup_default_session(profile_name='demo_cli')

client = boto3.client('s3')
```
