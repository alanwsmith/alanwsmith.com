title: Add A Slash Character To Cloudinary Text
slug: 20eOaV3hofgj--add-a-slash-character-to-cloudinary-text
---

use:

```
%252F
```

For example:

```
http://res.cloudinary.com/demo/image/upload/w_300,o_30/l_text:Arial_20_bold:www.cloudinary.com%252Fconsole/couple.jpg
```

---

Not sure why you'd need it, but an alternative is:

```
%E2%88%95
```

e.g. 

```
http://res.cloudinary.com/demo/image/upload/w_300,o_30/l_text:Arial_20_bold:www.cloudinary.com%E2%88%95console/couple.jpg
```
