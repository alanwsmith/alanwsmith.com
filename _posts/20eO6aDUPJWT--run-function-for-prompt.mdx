To run a function for a prompt, do something like:


function precmd() {

PROMPT="%~ $(ls -l | wc -l)
$ "

}


where `ls -l | wc -l` will be expaced. you have to do this in precmd() so that the code gets exectured each time. 


You can also put functions in there: 


function show_git_branch() {
    echo $(ls -la | wc -l)
}

function precmd() {

PROMPT="%~ $(show_git_branch)
$ "

}
