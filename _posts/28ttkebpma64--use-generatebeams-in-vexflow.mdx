The initial examples all show manually creating your own beam with like:

```javascript
const beam1 = new Beam([
    staveNotesUp[0],
    staveNotesUp[1],
    staveNotesUp[2],
    staveNotesUp[3],
])
const beam2 = new Beam([
    staveNotesUp[4],
    staveNotesUp[5],
    staveNotesUp[6],
    staveNotesUp[7],
])
```

But you can do this instead which is much nicer and automatically sends beams the proper direction:

```javascript
const beams = Beam.generateBeams(staveNotesUp)

// Formatter output of measure goes here. e.g.
// Formatter.FormatAndDraw(context, measure, staveNotesUp)

beams.forEach((b) => {
    b.setContext(context).draw()
})
```

Not sure why they don't show they in the initial example.
