This is with a spread and destructuring  

```javascript
function Box({className, ...rest}) {
    return <div className={`box ${className}`} {...rest} />
}
const element = (
    <Box className="blue" style={{fontStyle: 'italic'}}>
        This is the box
    </Box>
)
```

results in:

```html
<div class="box blue" style="italic">
    This is the box
</div>
```
