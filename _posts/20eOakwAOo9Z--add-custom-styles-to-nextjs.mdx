Do something like this in your `../styles/global.css` file:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  h1 {
    @apply text-2xl;
  }
  h2 {
    @apply text-xl;
  }
}
```


via: https://tailwindcss.com/docs/adding-base-styles
