Create a new [tmux](https://github.com/tmux/tmux/wiki) window with:

```command
control+b  c
```

That is: 

- Press and hold down the Control key then press "b" 
- Release both keys
- Then, press "c" 

Which is often abbreviated

```command
<C-b> c
```
