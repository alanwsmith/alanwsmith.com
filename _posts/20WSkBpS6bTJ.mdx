Use this in your statement

```sql
UNIQUE KEY (`streamerId`)
```

For example:

```sql
CREATE TABLE `statuses` (
  `primaryKey` int NOT NULL AUTO_INCREMENT,
  `lastUpdateMs` bigint NOT NULL,
  `streamerId` int NOT NULL,
  `currentSongId` varchar(30) NOT NULL,
  `currentSongDurationMs` int DEFAULT NULL,
  `currentSongProgressMs` int DEFAULT NULL,
  `nextSongId` varchar(30) DEFAULT NULL,
  PRIMARY KEY (`primaryKey`)
  UNIQUE KEY (`streamerId`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci;
```
