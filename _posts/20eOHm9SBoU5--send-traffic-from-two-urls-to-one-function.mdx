Just put a second decorator above the function with both (or, probably however many) routes like this:

```python
@app.route('/<snake>/end', methods=['POST'])
@app.route('/<snake>/move', methods=['POST'])
def move(snake):
    if snake == 'bravo_snake':
        print("Bravo Snake", file=sys.stderr)
        bs = BravoSnake()
        move = bs.get_move(move_request=request.get_json())
        return_data = { 'move': move }
        print(return_data, file=sys.stderr)
        return json.dumps(return_data)
```
