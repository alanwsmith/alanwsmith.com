Python ships with built-in functionality for generating output from templates that provide basic variable replacement. 

### From A String

```python
from string import Template

template = Template('Name: $name - Color: $color')

output = template.substitute(
            name="Karl", 
            color="Green"
        )

print(output)
```

Produces:

```txt
Name: Karl - Color: Green
```

### From A File

Assuming you have a files named `template.txt` with:

```txt
this is the name: $name
and the color: $color
```

Then this:


```python
from string import Template

with open('template.txt') as _tmpl:
	strcuture = Template(_tmpl.read())
	output = strcuture.substitute(
	   name="Karl",
	   color="Green"
    )
	print(output)
```

Will produce:

```txt
this is the name: Karl
and the color: Green
```
