I like clean URLs. Nothing more than, 

	http://example.com/page-name

---

	RewriteEngine On
	RewriteCond %{DOCUMENT_ROOT}/pages/$1.html  -f
	RewriteRule ^(.*)$ %{DOCUMENT_ROOT}/pages/$1.html [L]
