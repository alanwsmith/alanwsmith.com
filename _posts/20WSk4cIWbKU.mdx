Pull the page and parse it as JSON


```python
#!/usr/bin/env python

import requests

request = requests.get('https://api.weather.gov/stations/KSGJ/observations')
json_contents = request.json()
print(json_contents)
```

Pull the page directly as text

```python
#!/usr/bin/env python

import requests

request = requests.get('https://api.weather.gov/stations/KSGJ/observations')
contents = request.text
print(contents)
```
