The last part of a website's name (e.g. the ".com", ".edu", or ".uk") is called the Top-level Domain. Each country gets their own (e.g. ".au", ".uk", ".us"). There also used to be seven other "generic" possibilities:

1. .com (commercial)
2. .org (organization)
3. .net (network)
4. .int (international organizations)
5. .edu (U.S. education institutions)
6. .gov (U.S. government)
7. .mil (U.S. military)

That list has been growing in recent years. It's up to 810 at last count<sup>1</sup>. This makes it harder for cybersquatters<sup>2</sup> to snatch up all the good ones. There are simply too many possibilities to make it cost effective. For example, here's the current list of available my name dot whatever:
