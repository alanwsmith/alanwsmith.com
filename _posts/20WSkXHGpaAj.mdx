Do this to install it:

    pip install pyyaml
    
---

Example:


```python3
import yaml

with open('some_file.md') as md_file:
    md_data = md_file.read()
    md_parts = md_data.split('---')

    yaml_data = yaml.load(md_parts[1], Loader=yaml.FullLoader)
        print(yaml_data)
