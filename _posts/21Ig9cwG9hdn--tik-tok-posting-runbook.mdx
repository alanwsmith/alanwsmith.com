### Start

<Checklist data={`
Make new directory and put raw clip and premiere project in 'raw' folder
Import clip and drop it on the timeline
Set Audio Gain - Normalize Max Peak to -3dB
Apply toning presets 1a and 1b
`} />

### Cropping

<Checklist data={`
Set Sequence - Sequence Settings to 1080 horizontal x 1920 vertical
If a Delete All Previews warning dialog pops up, just click OK
Scale to zoom out to just before black bars show (possibly standard: 89% - Under Effect Controls -> Video)
Adjust Position to center things up (Under Effects Controls -> Video)
`} />

### Editing

<Checklist data={`
Edit In and Out Points
Clip video
Remove In and Out Points
TODO: See if you can just export with the in and out points marked?
`} />

### Export 

<Checklist data={`
Set Format to H.264 
Use "Match Source - High Bit Rate" or "Match Source - Adaptive High Bit Rate" (Adaptive is better if size works)
Keep Size below 300MB
TODO: Confirm size limit 
Set the filename and output location
In the Video tab check: Render at Maximum Depth
At bottom of the window, check: Use Maximum Render Quality
Click Export
`} />

### Publish

<Checklist data={`
Copy the file to: Dropbox/__tiktok
Wait...
On the phone open the video the do Share -> Export Video -> TikTok icon
Hit Next on the first page (assuming the clip is lined up)
Tap Captions and edit them
Add Text if something applies (~1.8 sec feels right for a default)
Hit Next
Fill out "Describe your video"
Possible hashtags #thepodofalan #art #creativity #workinginpublic #creativeprocess #creativepractice
Put a sticker on via "Select Cover" if one applies
Make sure Duet and Stitch are on if the video is short enough
`} />
