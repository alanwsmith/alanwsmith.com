title: Access S3 From Redshift With boto3

This allows Redshift to access all S3 buckets

IAM - Create Roll
- Click Redshift in the full list of services then click `Redshift - Customizable` under "Select your use case" for "Allows Redshift clusters to call AWS services on your behalf."
- In the permissions policy screen, select `AmazonS3FullAccess` then go to Next Tags
- Add any tags you'd like then click next. 
- Name the role: Redshift_S3_Read_Only_Access and change the description to note that it's for S3.
