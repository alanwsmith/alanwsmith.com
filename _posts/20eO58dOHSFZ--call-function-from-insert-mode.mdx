This is the way to call a function from insert mode. The key is `<cmd>`



    vim.api.nvim_buf_set_keymap(sbuf, 'i', '[', '<cmd>lua require"grimoire".change_selected_results()<CR>', {
        nowait = true,
        noremap = true,
        silent = true
    })
