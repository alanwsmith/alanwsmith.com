This probably isn't the best way to do it, but I current put plugin files in:

    ~/workshop/neovim-scratchpad/PLUGIN_NAME.lua
    ~/workshop/neovim-scratchpad/PLUGIN_NAME.vim 
    
And then run symbolic links in the main directories:

    cd ~/.config/nvim/lua
    ln -s ~/workshop/neovim-scratchpad/PLUGIN_NAME.lua
    
    cd ~/.config/nvim/plugin
    ln -s ~/workshop/neovim-scratchpad/PLUGIN_NAME.vim
